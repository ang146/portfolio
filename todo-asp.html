Todo App Description

Overview

This is a simple Todo application built using ASP.NET Core, Blazor and C#. The app allows users to add, edit and delete tasks from their todo list. Users can also mark tasks as completed and filter tasks based on their status.

Features

  • User authentication and authorization using Microsoft Identity.
  • Todo Tasks are user specific.
  • Add new tasks to the todo list.
  • Edit existing tasks.
  • Delete tasks from the list.
  • Mark tasks as completed or pending.
